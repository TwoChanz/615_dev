Tech Stack Guide 2026

The modern indie hacker stack for shipping fast and scaling smart

Philosophy

This stack prioritizes: developer experience, generous free tiers, proven reliability, and easy scaling. Every tool earns its place by reducing friction, not adding complexity.

The Core Stack

Framework Essential

Next.js 15+ (App Router)

Server Components, streaming, built-in API routes, excellent Vercel integration. The React ecosystem's clear winner for full-stack apps.

Alternatives: Remix (if you prefer loaders), Nuxt (Vue), SvelteKit (Svelte)

Language Essential

TypeScript

Catch bugs before runtime, better IDE support, self-documenting code. Non-negotiable for any serious project.

Styling Essential

Tailwind CSS + shadcn/ui

Utility-first CSS with copy-paste components. Ship fast without fighting CSS. shadcn/ui gives you accessible, customizable primitives.

Alternatives: CSS Modules, Styled Components (if you prefer CSS-in-JS)

Database Essential

Supabase (PostgreSQL)

Generous free tier (500MB, unlimited API requests), built-in auth, realtime subscriptions, auto-generated APIs. The Firebase alternative that doesn't lock you in.

Alternatives: PlanetScale (MySQL), Neon (serverless Postgres), Turso (SQLite edge)

Authentication Essential

Supabase Auth or Clerk

Supabase Auth: Free, integrated with your database. Clerk: Better UX, more providers, but paid after 10k MAU. Pick based on budget.

Alternatives: Auth.js (self-hosted), Lucia (lightweight)

Payments Essential

Stripe

Industry standard. Subscriptions, one-time payments, customer portal, invoicing. 2.9% + 30Â¢ per transaction. Excellent documentation.

Alternatives: Lemon Squeezy (handles tax/VAT), Paddle (merchant of record)

Hosting Essential

Vercel

Zero-config Next.js deployment, edge functions, preview deployments, generous free tier. The obvious choice for Next.js apps.

Alternatives: Netlify, Railway, Cloudflare Pages, Fly.io

Recommended Additions

Email Recommended

Resend

Developer-first email API. React Email for templates. 3,000 emails/month free. Simple pricing, great DX.

Alternatives: Postmark, SendGrid, AWS SES (cheapest at scale)

Analytics Recommended

PostHog or Plausible

PostHog: Full product analytics, session replay, feature flags. Free up to 1M events. Plausible: Privacy-focused, lightweight, EU-hosted option.

Alternatives: Mixpanel, Amplitude, Google Analytics (free but privacy concerns)

Error Monitoring Recommended

Sentry

Catch errors before users report them. Stack traces, breadcrumbs, release tracking. 5,000 errors/month free.

Alternatives: LogRocket (includes session replay), Bugsnag

File Storage Optional

Cloudflare R2 or Supabase Storage

R2: S3-compatible, no egress fees. Supabase Storage: Integrated with your database, simpler setup.

Alternatives: AWS S3, Uploadthing (simpler API)

Background Jobs Optional

Inngest or Trigger.dev

Serverless background jobs without managing infrastructure. Retries, scheduling, event-driven workflows.

Alternatives: Vercel Cron, BullMQ (self-hosted Redis)

AI/LLM Optional

OpenRouter or Vercel AI SDK

OpenRouter: Access multiple models (GPT-4, Claude, Llama) through one API. Vercel AI SDK: Streaming, hooks, edge-ready.

Alternatives: Direct OpenAI/Anthropic APIs, Replicate (for open models)

Monthly Cost Breakdown

What this stack costs at different stages:

Stage Users Estimated Cost
MVP / Side Project 0-100 $0-20/mo (free tiers)
Early Traction 100-1,000 $20-100/mo
Growing 1,000-10,000 $100-500/mo
Scaling 10,000+ $500+/mo (good problem to have)
Key Principle: Start with free tiers, upgrade only when you hit limits. Most indie projects never need to pay for infrastructure until they're profitable.

What I'd Skip

  • Kubernetes: Overkill for 99% of indie projects. Vercel/Railway handle scaling.
  • GraphQL: REST or tRPC is simpler for most apps. GraphQL adds complexity without proportional benefit.
  • Microservices: Start monolithic. Split later if you actually need to.
  • Self-hosted everything: Your time is worth more than saving $20/month.

Quick Start Commands

# Create Next.js app with TypeScript
npx create-next-app@latest my-saas --typescript --tailwind --app

# Add shadcn/ui
npx shadcn@latest init

# Add common components
npx shadcn@latest add button card form input

# Install essentials
npm install @supabase/supabase-js stripe zod react-hook-form
